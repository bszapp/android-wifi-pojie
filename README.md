# android-wifi-pojie
Android使用Shizuku授权的暴力破解wifi密码工具
Shizuku必须要以root身份启动（软件刚开发时用的是root过的手机，本来想这个命令adb权限就够了就用Shizuku了，没想到没root没法用命令连接）
