# v3.0.0 Alpha-01

只能使用Shizuku激活使用，不支持root、不支持无shizuku
主页点左上角菜单打开密码字典破解使用
先在密码字典破解设置里将所有项目改为带shizuku字样的，读取网络日志改为命令行后使用

目前支持：

- 手动输入密码列表运行
- 多任务排队运行、随时新增移除
- 保证ui与服务进度显示统一
- 后台运行（需要调整省电策略为无限制）
- 运行防弹窗密码错误（针对小米部分设备弹窗，运行时循环kill设置）
- 自定义应用主题色、深浅色（应用设置页修改）

不支持/未完成：

- root模式/普通模式
- 修改运行参数
- 除密码字典破解、实验室、设置以外的其他页面
- 密码本管理
